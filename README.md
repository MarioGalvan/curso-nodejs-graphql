### Install

```sh
yarn install
```


### Migrations

```sh
yarn run migrations:run
```

### Run in dev mode

```sh
yarn run dev
```

### Run in prod mode

```sh
yarn run start
```

### Up Docker database
```sh
docker compose up
```
